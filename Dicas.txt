Banco de dados
 1 - Instalando Laravel 
    composer global require "laravel/installer"

 -
 2 - Criar o projeto Laravel
    composer create-project laravel/laravel  todo --prefer-dist
 
 -
 3 - Model (para manipular os dados)
    Gerar arquivos em App
        php artisan make:model Tarefa
    
    OBS: Adicionar no arquivo App\Tarefas:
        protected $table = "tarefas";

 -
 4 - Criar o template layouts: app, menu e head
     Criar arquivo home.blade e alterar a rota

 -
 6 - Controller

    php artisan make:controller TarefaController --resource
    
    - Criar a rota 
        Route::resource('/tarefa','TarefaController');

 -
 7 - Views
    - Criar o arquivo create.blade

 -
 8 - Base de dados
    Arquivo .Env - Dados de conexao

 9 - Migration
    Gerar arquivos Migration
        php artisan make:migration tarefas_create_table --create=tarefas

    Criar tabelas no banco
        php artisan migrate
    
    Desfazer Migration
        php artisan migrate:rollback
        php artisan migrate:rollback --step=2


OBS: Pacote para gerar arquivo PDF (mPDF)
       - Instalar:
            composer require carlos-meneses/laravel-mpdf
       
       - Configuração do tamanho do papel é alterada em:
        \vendor\carlos-meneses\laravel-mpdf\src\pdf.php
            'format'   => 'A4',    
    
        - Depois adiciona app.php
            'providers' => [
                Meneses\LaravelMpdf\LaravelMpdfServiceProvider::class
            ]
        
            'aliases' => [
                'PDF' => Meneses\LaravelMpdf\Facades\LaravelMpdf::class,
            ]

        - use MPDF;
            function generate_pdf() {
                $data = [
                    'foo' => 'bar'
                ];
                $pdf = PDF::loadView('pdf.document', $data);
                return $pdf->stream('document.pdf');
            }
